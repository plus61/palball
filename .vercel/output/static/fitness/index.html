point 選ばれる5つの理由

reason1 衝撃的バリュープライス

日本屈指の衝撃的バリュープライス。6,980円(税込7,678円)で営業時間内はいつでもジム・スタジオ・プール・スパ・サウナの全エリア利用可能。さらに40歳未満なら3,980円(税込4,378円)！

reason2 初⼼者にやさしい

「ジムがはじめて」「どうやって運動したらいいか分からない」という方もご安心ください。ジムにはトレーナーが常駐しており、マシンの操作方法やトレーニング方法など丁寧にサポートいたします。地域No.1の「初心者へのやさしさ」を目指しています。

reason3 時短フィットネス

トレーニングマシンをフルリニューアル！安全性と効率性を兼ね備えた最新マシンで、少ないセットで効率的かつ効果的にトレーニングできます。

reason4 最新鋭のレッスンプログラム導入

初心者でも楽しめるレッスンを多数ご用意！海外で人気のあるレッスンや、トレンドを取り入れた最新レッスンも導入。アップデートがあって飽きない・楽しいプログラムを豊富に揃えています！

reason5 地域密着

湘南の茅ヶ崎で50年以上の歴史があるハヤシグループ（株式会社林水泳教室）が運営。おかげさまで、30年以上にわたり地域の方々にご愛顧いただいております。これからも地域の皆さまの健康づくりや仲間づくりの場となるサードプレイスをご提供していきます。

トレーニングジム

ノーチラスインパクトシリーズ

ノーチラスインパクトシリーズは、ヘビーウェイトスタックを標準装備しており、初心者からプロのアスリートまで幅広い層に支持されております。
スイッチのように重量を変更できる特許取得済みのLock N Loadウェイト選択システムを標準装備しております。

また、ノーチラス社創業者のアーサー・ジョーンズ博士が開発した「ノーチラスカム」を装備したトレーニングマシンは、トレーニングの最大効果を最大限に高めるための「回転抵抗」と「可変抵抗」を採用。筋肉にどの角度からでも安全に最大の負荷をかけられるため、少ないセット数で効率的な時短フィットネスが可能です！
日本最大級サプライヤー【think】監修オリジナル使用動画が付いているので 安心して最大の効果を得られます。

スタジオプログラム

アクアプログラムはバランスよく身体を鍛えられる全身を使った有酸素運動です。
地上で行う運動に比べ、ひざや腰に負担が少ないため、痛みの気になる方も運動を続けられます。
水中ウォーキングやアクアビクスなら泳ぎが苦手な方でも気軽に楽しく身体を動かせます。

スイミング

ウォーキング

水の中で歩行運動を行います。目的に合わせた基本的な歩き方が中心で、初めての方でも安心のプログラムです。

クレジットカードで決済

会員情報を登録すると、初期費用決済の手続き画面へ移行します。
お手持ちのクレジットカード情報をご入力いただくことで、初期費用の決済と会費の引き落としをするクレジットカードの登録が完了します。

  • ※ デビットカード、プリペイドカードはご利用いただけません。
  • ※ [pal-ball@hsc-g.co.jp]からのメールを受信できるよう、設定をお願いいたします。
  • ※ URLリンク付きメールを受信できるよう、設定をお願いいたします。

ご来店の上、顔認証登録と誓約書への署名・ガイドブックの受け取り

ご来店時には必ず公的身分証明書（現住所と一致するもの）をお持ちください。当日は入館に必要な顔写真の撮影を行い、誓約書へのご署名をいただきます。手続き対応時間内にご来店ください。

  • ※ 最終の来店可能時間は手続き対応時間終了の30分前までとなります。

手続き対応時間

  • 月～金曜日 ：10:00～21:00　※水曜日除く
  • 土曜日 ：10:00～19:00
  • 日曜・祝日 ：10:00～18:00
